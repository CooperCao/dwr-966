Konfiguracja APN

Konfiguracja APN wyświetla aktualnie używaną listę profili.
Wibierz jedną pozycję z listy i kliknij Połącz aby ustanowić połączenie.

Aby edytować profil:
1. Wybierz profil i kliknij Edytuj.
2. Wprowadź informacje dostarczone prze operatora sieciowego.
3. Kliknij Zastosuj aby aktywować zmiany.
