Statyczny routing IPv6

Routing statyczny pozwala na ręczne określenie reguł dostarczania pakietów poprzez określenie sieci docelowych i odpowiadających im bram, przez które ruch ma być do tych sieci dostarczany. Tablica pokazuje listę wszystkich statycznych tras IPv6 i pozwala na ich edycję i usuwanie.

  » Nazwa: Nazwa trasy routingu.
  » Docelowy adres IP: Docelowy host lub sieć.
  » Prefix: Długość prefiksu do którego wiedzie trasa routingu.
  » Sieć: Sieć do której należy trasa routingu.
  » Brama: Adres IPv6 bramy przez którą docelowy host lub podsieć są osiągalne.

Kliknij Dodaj aby dodać trasę statyczną lub wybierz element i kliknij Edytuj aby ją zmodyfikować. Kliknij Usuń, aby skasować wybrany element.
