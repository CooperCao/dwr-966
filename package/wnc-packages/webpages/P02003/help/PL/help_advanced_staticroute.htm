Zaawansowane
Routing statyczny

Routing statyczny to metoda wybierania jednej z wielu istniejących dróg dostarczania paskietów w sieci. Tablica prezentuje wszystkie zdefiniowane trasy statyczne.
  » Nazwa: Nazwa trasy routingu.
  » Sieć docelowa: Adres sieci lub hosta docelowego, do któego prowadzi trasa routingu.
  » Maska podsieci: Maska podsieci dla trasy docelowej.
  » Sieć: Wyśiwetla sieć, do której należy trasa routingu.
  » Brama: Adres IP bramy przez którą docelowy host lub podsieć są osiągalne.

Kliknij Dodaj aby dodać trasę statyczną lub wybierz element i kliknij Edytuj aby ją zmodyfikować. Kliknij Usuń, aby skasować wybrany element.
