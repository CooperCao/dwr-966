D-Link DWR-966
System
Uaktualnienie firmware

Firmware będzie uaktualniany przez wprowadzenie nowych funkcji lub usunięcie wykrytych błędów. Ta sekcja umożliwia uaktualnienie oprogramowania urządzenia do nowej wersji.
Wybierz, aby uaktualnić firmware dla całego systemu lub modułu LTE:
1. Kliknij Przeglądaj aby wybrać plik firmware.
2. Kliknij Uaktualnienie routera/Uaktualnienie modułu aby rozpocząć proces uaktualniania.
