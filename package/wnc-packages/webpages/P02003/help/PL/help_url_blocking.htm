D-Link DWR-966
Zaawansowane
Blokowanie URL

Blokowanie URL pozwala blokować witryny internetowe, których adresy zawierają określone słowa kluczowe. Poniżej wyświetlona jest lista reguł blokowania:

  » Nazwa filtra: Unikalna nazwa reguły.
  » Słowa kluczowe: Słowa kluczowe użyte do filtrowania.
  » Domeny: Domena internetowa do której odnoszą się reguły filtrowania.

Kliknij Dodaj aby dodać regułę blokowania lub wybierz element i kliknij Edytuj aby ją zmodyfikować. Kliknij Usuń, aby skasować wybrany element.
