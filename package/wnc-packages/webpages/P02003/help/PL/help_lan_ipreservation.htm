D-Link DWR-966
LAN
Rezerwacja DHCP

Adres IP może być zarezerwowany dla konkretnego komputera. Umożliwia to urządzeniom takim, jak drukarka lub telefon VoIP na otrzymywanie niezmiennie tego samego adresu IP przydzielonego z serwera DHCP.

  » Lista zarezerwowanych adresów IP:
  »» Adres MAC: Adres MAC komputera lub innego urządzenia.
  »» Adres IP: Adres IP, który ma być przypisany temu urządzeniu.

Kliknij Dodaj aby dodać rezerwację IP lub wybierz element i kliknij Edytuj aby ją zmodyfikować. Kliknij Usuń, aby skasować wybrany element.
