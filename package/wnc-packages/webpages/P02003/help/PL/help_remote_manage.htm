D-Link DWR-966
System
Zdalne zarządzanie

Ta funkcja pozwala zdalnie zarządzać urządzeniem poprzez port WAN. Domyślnie, zdalne zarządzanie jest wyłączone. Aby włączyć zdalne zarządzanie, zaznacz pole wyboru.
UWAGA: Kiedy zdalne zarządzanie jest włączone, urządzenie jest dostępne dla każdego, kto zna jego adres IP. Zalecane jest, aby przed włączeniem zdalnego zarządzania, zmienić hasła administratora i użytkownika.
Zdalne zarządzanie może być skonfigurowane poniżej:
  » Włącz zdalne zarządzanie: Zaznacz to pole aby włączyć zdalne zarządzanie.
  » Typ dostępu: Wybierz typ dostępu. Jeśli wybierzesz Wszystkie adresy IP, urządzenie będzie dostępne z każdego adresu IP. Jeśli wybierzesz Zakres adresów IP, będzie konieczne określenie początkowego i końcowego adresu IP zakresu. Jeśli wybierzesz Tylko ten komputer, będzie konieczne wprowadzenie adresu IP komputera mającego zezwolenie na dostęp do zdalnego zarządzania.
  » Adres IP: adres IP komputera mającego zezwolenie na dostęp do zdalnego zarządzania

  » Konfiguracja zakresu adresów IP
  »» Początkowy adres IP: Początkowy adres IP dla zakresu.
  »» Końcowy adres IP: Końcowy adres IP dla zakresu.
