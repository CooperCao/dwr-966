Wi-Fi
Wi-Fi Protected Setup (WPS)

  » Wi-Fi Protected Setup (WPS): WPA jest standardem łatwej konfiguracji zabezpieczeń sieci bezprzewodowej.
1. Zaznacz aby włączyć WPS.
2. Wybierz SSID sieci bezprzewodowej do której chcesz ustanowić połączenie WPS.
3. Kliknij Zastosuj aby aktywować zmiany. Kliknij Zresetuj aby anulować zmiany.

  » Połączenie przez naciśnięcie przycisku (PBC): Jeśli inne urządzenia bezprzewodowe obsługują metodę WPS PBC, kliknij "Kliknij PBC", a następnie naciśnij przycisk WPS w drugim urządzeniu.

  » Osobisty Numer Identyfikacyjny (PIN): Jeśli inne urządzenia obsługują metodę WPS PIN, wprowadź taki sam PIN w urządzeniu klienckim i kliknij ikonę PIN aby ustanowić połączenie.
