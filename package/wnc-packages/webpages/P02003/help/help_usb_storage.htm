Advanced
SAMBA Server Settings

SAMBA is the protocol by which a lot of PC-related machines share files and printers and other information such as lists of available files and printers.

  » Before designating a shared folder, you will need to enable the SAMBA server function:
  »» Enable SAMBA: Select this check box to enable SAMBA.
  »» SAMBA Server Name: The maximum server name length is limited to 15 characters. Users can browse the shared folders by the SAMBA Server Name.
  »» User Account: The account for accessing the SAMBA server. Current user accounts are "admin" and "all". The "all" user account does not require a password; the "admin" user account requires a User Password.
  »» User Password: The password of the "admin" user account used to access the SAMBA server.
  »» Display Password: Displays the User Password when the check box is selected.

  » To designate a shared folder for file sharing:
You can designate a folder within the USB storage device to serve as a shared folder. Computers connected to the DWR-966 will then be able to browse the shared folder by typing "\\[DWR-966's LAN IP address]\[Sharing Profile Name]" or "\\[SAMBA Server Name]\[Sharing Profile Name]" in your browser's address field.
  »» 1. Insert the USB storage device into the USB port. The information of this USB storage device will be listed in the table at the top of the "USB Storage" page.
  »» 2. Click the Add button to display the screen for adding/modifying a sharing profile.
  »» 3. The file folders of the connected USB storage device will be listed in the table at bottom of this page. Select the folder that you want to designate as the shared folder. The Folder Path field will automatically populate with the selected path.
  »» 4. Enter the information for this sharing profile and click the Apply button to implement this sharing profile.
